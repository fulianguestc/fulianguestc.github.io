Linux命令

关于档案与目录的一些基本命令

档案与目录管理

cd（变换目录）

. 代表此层目录
.. 代表上一层目录
- 代表前一个工作目录
~ 代表【目前用户身份】所在的家目录

pwd（显示目前所在的目录）

-P 显示确切的目论

mkdir（建立新目录）

-p 建立多层目录
-m 建立带有权限的目录

rmdir（删除【空】的目录）

-p 连同上层空的目录一起删除

ls（档案与目录的检视）

-a 全部的档案，连同影藏档
-d 仅列出目录本身
-l 档案的属性与权限

cp（复制档案或目录）

-a 复制所有属性（一模一样）
-i 若目标文件已经存在时，在覆盖时会先询问动作的进行
-p 连通档案的属性一起复制过去，而非使用默认属性
-r 递归持续复制，用于目录的复制行为

rm（移除档案或目录）

-f 忽略不存在的文档，不会出现警告讯息
-i 在删除前会询问使用者是否动作
-r 很危险，会把该目录下所有内容都删除

mv(移动档案与目录，或更名)

-f 如果目标档案已经存在，不会询问而直接覆盖
-i 若目标档案存在，就会询问是否覆盖
-u 若目标档案存在，且source比较新，才会更新

cat（档案内容查阅） tac（反向列示）

-b 列出行号，空白行不标行号
-n 打印出行号，连同空白行也会有行号，与-b不同

档案权限

chgrp（改变档案所属群组）

要改变的组名必须要在/etc/group档案内存在才行，否则就会显示错误

-R 进行递归的持续变更

chown（改变档案拥有者）

-R 进行递归的持续变更

chmod（改变权限）

xyz 数字类型的权限属性，为rwx属性值的相加
-R 进行递归(recursive)的持续变更
