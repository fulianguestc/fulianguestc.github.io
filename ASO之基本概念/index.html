ASO之基本概念

  • ASO
  • 产品策划/运营
工作日志

总结自公众号【姑婆那些事儿】ASO优化系列，并融入自己的思考——

ASO是什么？

ASO(全称App Store Optimization)是指应用商店的搜索优化，是针对你的APP所在的APP市场（国内仅限于apple store）进行搜索入口的数量及搜索结果排名提升的过程。

为什么要做ASO？

App Store搜索是其实是最高效的流量来源。研究表明73%的移动应用程序安装行为都是通过从App Store中进行搜索而获得的。伴随着其它渠道的疲软，ASO重要性愈加凸显。对于创业型公司，ASO可能是为数不多小投入大产出的推广手段。如果你的应用覆盖足够多的热词，并且排名靠前，带来的用户还是非常可观的。

ASO指数是针对应用对于苹果商店的搜索评估值，结合可见搜索词汇，搜索热度，搜索权重，搜索排名以及多个算法交叉计算的结果。可以根据ASO Index的值来判断应用在苹果商店的优化程度，ASO指数的值越高，证明该应用在苹果商店的搜索优化做的越棒。

ASO的目的是什么？
  1. 提高热词（可以在app store中通过搜索找到你App的词）的覆盖：增大app的品牌曝光，带来更多的自然用户。
  2. 提高热词搜索结果排名：排名靠前，增大app的下载转化。一般排名前三位会瓜分这个热词70%以上流量，所以我们目标就是进入热词搜索排名前三位；这对于需要融资的创业团队而言，也可以讲一个好的故事。
  3. 精准定位目标用户：提高app的用户活跃度。
  4. 维持排名：目前app运营推广效果维持，搜索排名是维持和提高的唯一可行途径。
  5. 提高app权重值：每个app都有一个权重值，这是科学ASO的参考依据。
哪些因素影响ASO？
  1. 应用名称：包含主标题和副标题，好的标题可以事半功倍。
  2. 关键字或者标签：总计100个字符，优秀的ASO每个字符平均承担5-10个热词覆盖空间。
  3. 用户评论：如果有1个一星差评，需要找补好评40+。
  4. 排名下载：与ASO存在互促关系，但不是正比，并非多多益善。
  5. 用户活跃度：好产品不是万能的，产品不好是万万不能的。
如何评估ASO的效果？
  1. 热词覆盖变大：优化前和优化后对比。每个版本应该会进步。
  2. 热词排行提升：优化前和优化后对比。每个版本应该会进步。
  3. 下载量变化：在同样的前提下，优化前和优化后对比。请注意，优化前后的情况相同的前提，最好是投放前不推广时期的量和只做ASO后的量前后对比（时间越长应该越明显），如果优化期间照常推广，那么优化前后app的排名曲线波动变化可以作为参考指标（可以借助相应的历史排名对比工具），一般波动越小效果越好这也是ASO维持推广效果的作用。
  4. 此外还有新增激活率，用户留存率，用户活跃度等内部参考指标。
CP（Content Provider移动数据内容提供商）要做的是什么？
  1. 知道自己产品目前有哪些热词，这些热搜词的对产品的影响，通过该词搜到及排名（需要借助热词查询工具）。
  2. 知道竞品有哪些热搜词，排名如何？
  3. 哪些词想覆盖，覆盖的方向是什么？
  4. 找个好渠道，好好配合，多沟通。与合作ASO的代理保持紧密的沟通，随时优化。
  5. 尽量遵守方案。对于合作ASO的代理要充分信任，双方还要多沟通。
IOS8热门搜索推荐词

影响因素主要是单位时间内单一用户搜索行为的累积。约95%以上的推荐为热词，覆盖相关的推荐热词可以在推荐期间获利。但是前提是我们自己如果不做热门推荐，那么我们的ASO要做到足够全的相关覆盖及足够多的覆盖排在前十才能坐收竞品渔利。
