XML和JSON简介

  • JSON
  • XML
Android

通常情况下，每个需要访问网络的应用程序都会有一个自己的服务器。我们可以向服务器提交数据，也可以从服务器上获取数据，所以一般我们会在网络上传输一些格式化后的数据，这种数据会有一定的结构规格和语义，当另一方收到数据消息之后就可以按照相同的结构规格进行解析，从而取出想要的那部分内容。在网络上传输数据时最常用的格式有两种，XML和JSON，接下来我们就学习一下这两种方式的解析方法。

XML

解析XML格式的数据有很多种方式，下面介绍比较常用的两种，Pull解析和SAX解析。

Pull解析方式

XmlPullParserFactory factory = XmlPullParserFactory.newInstance();
XmlPullParser xmlPullParser = factory.newPullparser();
xmlPullParser.setInput(new StringReader(xmlData));
int eventType = xmlPullParser.getEventType();//得到当前的解析事件
String id = "";String name = "";String version = "";
while(eventType != XmlpullParser.END_DOCUMENT){
    String nodeName = xmlPullParser.getName();
    switch(eventType){
        //开始解析某个节点
        case XmlPullParser.START_TAG:
            if("id".equals(nodeName)){
                id = xmlPullParser.nextText();
            }else if("name",equals(nodeName)){
                name = xmlPullParser.nextText();
            }else if("version",equals(nodeName)){
                version = xmlPullParser.nextText();
            break;
        //完成解析某个节点
        case XmlPullParser.END_TAG:
            break;
        default:
            break;
    }
    eventType = xmlPullParser.next();
}

SAX解析方式

SAX解析是一种常用的XML解析方式，虽然它的用法比Pull解析要复杂一点，但在语义方面会更加清楚。通常情况下我们会新建一个类继承DefaultHandler，并重写父类的五个方法。

  • startDocument()方法会在开始XML解析的时候调用。
  • startElement()方法会在开始解析某个节点的时候调用。
  • characters()方法会在获取节点中内容的时候调用。
  • endElement()方法会在完成解析某个节点的时候调用。
  • endDocumnet()方法会在完成整个XML解析的时候调用。

具体代码

SAXParserFactory factory = SAXParseFactory.newInstance();
XMLReader xmlReader = factory.newSAXParser().getXMLReader();
ContentHandler handler = new ContentHandler();//上文提到的新建类
//将ContentHandler的实例设置到XMLReader中
xmlReader.setContentHandler(handler);
//开始执行解析
xmlReader.parse(new InputSource(new StringReader(xmlData)));

解析JSON格式数据

比起XML，JSON的主要优势在于它的体积更小，在网络上传输的时候可以更省流量，但缺点在于，它的语义性较差，看起来不如XML直观。

使用JSONObject

JSONArray jsonArray = new JSONArray(jsonData);
for(int i=0; i<jsonArray.length(); i++){
    JSONObject jsonObject = jsonArray.getJSONObject(i);
    String id = jsonObject.getString("id");
    String name = jsonObject.getString("name");
    String version = jsonObject.getString("version");
}

使用GSON

GSON可以将一段JSON格式的字符串自动映射成一个对象，不需要手动去编写代码进行解析。
例如：一段JSON格式数据如下：

{“name”:"Tom", "age":20}

那我们可以定义一个Person类，并加入name和age两个字段，然后只需简单地调用如下代码即可将JSON数据解析成一个Person对象：

Gson gson = new Gson();
Person person = gson.fromJson(jsonData, Person.class);

当解析一段JSON数组会稍微麻烦一点，需要借助TypeTolen将期望解析成的数据类型传入到fromJSON()方法中：

List<Person> people = gson.fromJson(jsonData, new TypeToken<List<Person>>(){}.getType());
