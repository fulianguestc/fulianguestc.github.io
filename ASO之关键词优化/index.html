ASO之关键词优化

  • ASO
  • 产品策划/运营
工作日志

总结自公众号【姑婆那些事儿】ASO关键词优化系列，形成关键词干货篇，并融入自己的思考——

输出结果：本文以及拾光官网www.keshile.net

一、什么是关键词

关键词即AppStore搜索引擎收录用户频繁搜索的索引词。
对于ASO关键字调查，至今为止，尚未有一款工具把开发者从无聊并庞大的调查中解放出来（目前对应用的关键字调查并没有好工具）。

AppStore搜索引擎对于关键词权重的排序：app name > app keywords > app Description > IAP item Name/Description。

如何做100个字符(即app keywords)，附100字符监测工具。

二、关键词的作用？

关键词使App更容易被AppStore搜索引擎收录，在关键词搜索结果中显示App。本文介绍如何利用100个字符，做出更多的关键词覆盖，增加App的曝光机会。
没有在关键字上花时间，这会直接导致损失大量潜在下载量。

三、如何评估关键词运营效果？
  • KCN(Keywords Covering Number),即关键词有效覆盖个数。100个字符被AppStore搜索引擎分割出尽可能多的关键词数目，即KCN值越大越有利。
  • iAR(iOS App Rank for iTunes)，即IOS App权重，会影响到App的ASO情况，包括搜索排名、搜索联想排序等，其值跟下载量、评论、榜单排名、关键词覆盖数目、关键词搜索排名等相关，KCN的优化也能增大iAR值。
四、热词/关键词选择原则——选择流量大、并且竞争难度小的热词。

一些ASO的基本术语：

  1. 关联性（Relevance）：某个特定的关键字与你的应用以及目标用户之间的相关性，只有具有关联性的关键字才能产生有效转化。
  2. 难度（Difficulty）：某一特定关键字的竞争激励程度，这个值越高意味着它更难进入前列排名。（容易被挤下去了）
  3. 流量（Traffic）：特定关键字的抢手程度，在搜索中被搜索的次数越高，这个值就越高（热度）。

（一）获取关键字数据

现在，你已经有了一个关键字字库，之后你必须获得特定的应用商店数据，许多ASO工具都会为你呈现关键字流量和关键字难度。这个步骤很关键，之后，你需要获得每一个关键字的流量和难度，这一点非常重要，因为在进入第二步之前，你需要把关键字库中那些零流量的关键字全部清空（或者把他们放到一个待定区，因为有些关键字很有可能在未来一段时间内再次获得流量，谁能想到flappy bird突然能红起来呢？）。根据你的游戏类型，以及你采用ASO工具采集到的信息，平均一下iPhone和iPad的得分，从而获得一个合计难度/流量数字。

（二）明智地选择你的关键字

Appstore对于关键字列表的长度限制在100个字符以内，在这个有限的长度内，我们需要决定到底使用哪些关键字。目的是从关键字库中选择适当的关键字，带来有效的下载量。首先，我们要记住：

高相关性+适当的难度+一般的流量=下载量的增加。

分析：

  • 适当的难度：我们需要一些排名不算太困难的关键字，因为我们的产品本身在Appstore上基本上是没有知名度的。
  • 一些流量：我们需要有一定搜索量的关键字，否则我们会浪费100字符的空间，没有流量就不会带来下载

（三）选择关键字的难度

如何选择竞争难度小的热词？

竞争难度大小，就得看商务人员对整个行业竞品的了解程度了，需要你每天观察各个积分墙上竞品都在疯狂做哪些词，尽量避开这些热词，优先选择竞争难度小的词来做。

为关键字打分。为你所有有流量的关键字做一个排名，并为他们的难度分一下等级。

排名依据有：

  • 总下载量（当前版本和以往版本）
  • 下载动量（下载加速度）
  • 评论量（正反面评论和历史评论）
  • 玩家活跃度（应用被打开的频率）
  • 抛弃率（应用被删除的比率）
  • 点击下载率（有多少下载量来自于搜索下载）

一般来说，我们的应用中排名前十的有流量的关键字需要大量的测试和评估，所以不要期望一次就选中你想要的关键字。ASO优化需要时间，需要持续的监控和调整，这是一个不断发展的过程。

定期修改关键字。因为Appstore会经常改变他们的算法，所以我们需要有快速反应和修改的能力。选择一个高相关、低难度的关键字以及至少有一些搜索量的关键字是你最好的而选择。每一次更新关键字列表都会给你带来更多的收获，所以第一次提交后要放弃，保持筛选的过程。

（四）难度等级

如果你的软件已经上线，那么现在可以分一下关键字排名列表了，理想情况下，你应该随时关注关键字对下载所造成的影响。这个信息很有价值，因为它会帮助你对于后续的决策作出很好的指导作用。知道你的关键字排名对于你确定之后的难度等级也是一件很重要的事，分出难度等级可以帮你快速缩小潜在的关键字列表。

我们排列出以下难度等级：

  1. 目标关键字：（难度小于2，以及排名小于10），这是你的首选，这种关键字让你有机会出现在搜索的前十位中；
  2. 备选关键字：（难度在2-4之间，以及排名在10-30之间）这是你的备选，如果你不能确定足够多的目标难度关键字，那么你可以用这些关键字填满你的列表；
  3. 排除关键字：（难度大于4，排名大于30）：这是你需要排除的关键字。这些关键字几乎没有机会帮你带来下载量。

如果没有足够的备选关键字——理想状态下，你应该拥有足够多的目标关键字可以选择。如果没有足够的备选关键字的话，建议重新读一次第一篇文章。游戏上架才是真正的开始，从游戏上架的那一刻，你就应该时刻追踪关注你的关键字排名情况，关注评论、排名、下载等元素，确定你到底需要哪些关键字。

（五）流量为王

没有流量就没有一切。在有了一定流量的情况下，我们会更多关注难度和关联度。比方说你只有一个关键字可以填写了，但是你还有三个潜在关键字可以做选择，如果他们都是高关联度和相同难度的话，那么自然要选择高流量的关键字。

如何选择流量大的热词？衡量流量大小的方法：

  • 通过搜索词热度（app热度查询网址）来看，比如热词A热度为5000，热词B热度为7000，那么热词B流量更大。（热度越高+相关=流量越大）。
  • asou的“asou趋势-搜索排行“可以看到各个分类中的热词分布，对于优化关键词也有一些帮助，有兴趣的可以研究一下。一般优化上去的热词，在没有竞品竞争的情况下，排名一般不会有太大波动，可谓一劳永逸吧。
  • 选定了一些主词后，在appstore搜索这个词，下拉联想词中的词也很有参考价值。也可以辅助百度指数来验证流量大小（但百度指数主要是web端，web端搜索行为和移动端还是有一些差别的）。

验证效果：可以通过appannie，和asou（ios的）最新推出的ASO指数来验证，指数越高说明ASO做的越好。

Tips：

  • 注意关键字的长度：尽量保证每一个关键字小于10个字符。
  • 保持快速更新：因为每次更新，你都将有机会删除表现不佳的关键字以及增添新的关键字。
五、关键词的做法

怎么做ASO，需要做哪些，流程如何？总体而言：

  1. 研究目前状态：产品功能是什么？确定用户群体是什么？
  2. 这部分用户群体导出来，为这部分人群匹配相应的用户搜索行为的关键字。
  3. 为相应的用户搜索行为关键字匹配关联热词（收集数据越多越好，数据可参考apple store的联想词或相关工具的数据）。
  4. 把这部分词筛一遍，找出最有效率的集合，提取集合中的有效字符分拆组合（热词不拆分有苹果的审核风险，拆分过细可能无法覆盖，要掌握度）。

具体步骤如下：

  • 第一步：知己知彼，建立一个关键字库。

（一）判断具有关联性的关键字
记住要使用那些跟你的应用和用户有高关联度的关键字。有些关键字看起来有着排名优势，可是对于你的产品来说根本没有意义，用户即使搜到了也不会下载。

（二）分析竞争对手的关键字
附：AppYing提供的ASO利器，其功能：

  • 基于APP现状和大数据分析，我们提供给您合理的关键词优化建议，在您设置好新版关键词（提交到iTC后台）并上线后，我们将持续跟踪评估优化效果（每周跟踪一次），并结合数据给出进一步的优化建议。
  • 可以定制添加关键词，查看追踪关键词的搜索热度、搜索结果，以及近期变化情况，并且智能解析相关联词汇。
  • 实时跟踪应用在关键词下的排名，以及该词的搜索结果，有助于开发者查看关键词数据及时迭代关键词。
  • 结合App属性，人工智能推荐Keywords，提供Keywords优化解决方案与技术支持服务。

我们可以通过分析竞争对手的情况来判定哪些关键字具有关联性。

  1. 了解自己的潜在用户和竞争对手是谁。你可以在Appstore排行榜内寻找自己的竞争对手。当然这里有个小诀窍，你不用一个个浏览不同的热门应用，只要使用App Figures的Top Apps工具就行，它可以极为便利地为你展示多达三个分类的热门排名应用。除此之外，你还可以查找Appnique的“推荐竞争对手”列表，以及Sensor Tower的“追踪竞争者”工具。
  2. OK，现在你已经找到你的竞争对手了，利用Sensor Tower的Keywod Spy可以让你发现竞争对手的关键字，这个工具除了让你发现竞争对手的关键字以外，还能让你看到自己产品的关键字与之重叠的情况。
  3. 最后，制作一个广泛的、含有潜在关联性的关键字库，不同的ASO工具可能会给你提供不同的关键字，而你的工作就是手动筛选出最终可以使用的那些关键字。

（三）判断关键字的其他办法

  1. 软件名称。许多应用软件在自己的名称中其实就会含有一些重要的关键字，所以名称是一个很好的切入点，总体来说，你应该在你的应用中添加一些高流量、高难度的关键字。
  2. 软件描述。仔细观察竞争对手的应用描述，有些应用很可能会在描述中包含潜在关键字。不过值得注意的是，Appstore（ios）的算法可能不会将应用描述中的文字列入排名考虑因素。
  3. 软件评价。竞争对手产品的高分评价中有时候也会有潜在关键字，从五星评价开始找，因为五星评价是最直观的玩家的需求（小雨：对产品设计也需要重视五星玩家/铁粉的需求）。
  4. 谷歌趋势。如果你想知道某个关键字的检索量，其实谷歌趋势也是个不错的选择。依旧以数学游戏作为例子，我们把“数学游戏”作为基础关键字，同时查询“数学运动”和“数学训练”进行对比。对于“数学运动”感兴趣的检索几乎为0，这表明这很有可能是一个冷关键字，除此之外，谷歌趋势还会为你提供一些关键字建议。（小雨：需翻墙）。

提高KCN值的同时，更需要寻找符合自己的好词，有些热词虽然搜索有排名，但是排名100名开外，没有实际价值。

怎么找合适的关键词？——关键词拓展

想做到喜马拉雅那种地步，还需要做到以下两步——

  1. 更新版本勤快些，有事没事更新下，其实还能获得苹果官方的流量福利；
  2. 对比版本更新的不同，跟踪每个关键字的变化，及时剔除差的关键字，测试新的关键字。不断迭代测试，做100字符优化是门修行。
六、操作实例示范

以酷我K歌举例，带大家一步一步进行实际操作：

  1. 首先利用网址”3“，输入你的app的名字，并记录下你目前已有的关键字。记录下当前你的app的关键字及排序情况。比如，当时我们酷我K歌当时只有9个关键字，分别是：ktv、酷我、酷我音乐、唱歌、音乐、电台、酷我音乐、唱吧、k歌。
  2. 了解3-4个同行业竞品的关键字分布情况。比如把K歌行业的唱吧、爱唱、其他2-3个app的关键字情况同时记录下来——找出他们这些app都重合的关键字及排名前三的关键字，进行对比——结合整个K歌行业的分布情况（唱吧最大，所以我肯定格外“照顾”），估算总结出竞品的最有效的关键字排序情况。由此，我得到的是：音乐 爱音乐 唱歌 爱唱 唱吧 手机ktv 视频 k歌 ktv 电台 直播。
  3. 寻找潜在的关键字。这一步需要分析下当前K歌行业的关联搜索词，并且预判未来可能火的一些词，同时结合时下正在热搜的关键词（需要用到之前的第一个工具，最好是和自己行业相关的词，不能乱加的）。比如，K歌时可能搜索的词有：练歌、直播、伴奏、歌词、练唱图等等；时下正在热搜的词：多听fm、女神、求合体2（可查到关键词排行榜）。预判可能火的词：练歌。
  4. 综上，预估并总结出最终的20-30个左右的关键字。将以上的所有词筛选、合并归整、仔细斟酌，最终我得出的结论，按照排序依次是：音乐,酷我,唱歌,唱吧,爱唱,酷我音乐,酷狗音乐,手机ktv,酷我K歌,爱音乐,酷狗,电台,k歌,K歌达人,视频,秀场,直播,练歌。

Tips：

  1. ASO的关键字设置不是一成不变的，需要不定期更换；
  2. 预判能力以及对行业的理解和观察，需要平时的积累；
  3. 规范化：关键字必须跟自己的app相关，和seo一样，可能会被视为作弊；
  4. 太热门的词，比如微信、qq、淘宝等这种，我都试过，一点用都没有。大家都加，等于大家都不加，就算有用，轮到你也没几个量了。
  5. 懂SEO知识的人做ASO会相对容易，逻辑差不多；
  6. 搜索结果可能跟你设置的关键字不一致，没关系。
  7. 根据搜索结果做出相应调整，但某些词一看就临时出来的一些，比如“陪我学”，过一阵子肯定会消失，所以我是不可能添加至关键字设置里的。
  8. 注意关键行业品牌词。比如竞品词，如果做到第一，会被投诉导致苹果警告（最新的搜索规则更改后，涉及品牌的竞品词几乎不存在这种机会了），一般排前5都会存在风险。当然如果不涉及品牌的热门相关词，完全不用担心优化的风险。
